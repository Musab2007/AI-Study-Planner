# AI Study Planner + CS Career Helper

## Overview
AI Study Planner + CS Career Helper is a beginner-friendly project designed to help students organize study time, understand basic computer science topics, and track important academic goals.

## Features
- Weekly study planner
- AI-style beginner explanations
- Goal and deadline tracker
- Clean web interface

## Tools Used
- Google Sites
- Google Docs
- GitHub
